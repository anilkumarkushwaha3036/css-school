##  The Complete 7-Module Curriculum

### 01 Box Model Lab
* **Scope**: `padding`, `margin`, `border`, and `box-sizing` modes (`border-box` vs `content-box`).
* **Interactive Tooling**: Nested diagram values updating dynamically on slider input; hover highlights mapped to preview segments.

### 02 Flexbox Arena
* **Scope**: Direction flows, main/cross-axis alignment (`justify-content`, `align-items`), wrap layers, item spacing (`gap`), and selective item overrides (`flex-grow`, `flex-shrink`, `align-self`).
* **Interactive Tooling**: Sandbox allowing insertion/removal of up to 6 flexible cards with independent settings.

### 03 Positioning Lab
* **Scope**: Viewport document flows comparing `static`, `relative`, `absolute`, `sticky`, and `fixed` layering offsets.
* **Interactive Tooling**: A simulated scrolling viewport where headers stick to frame borders, parent containers constrain absolute items, and fixed cards float globally.

### 04 Grid Space
* **Scope**: Grid tracks, fractional units (`1fr`), columns repeating rules, and track line indicators.

### 05 Styles & FX Lab
* **Scope**: Color synthesis (HSL hue loops), linear gradients angles, `border-radius`, solid borders, and outer or `inset` drop shadows.

### 06 Typography Studio
* **Scope**: Outfit, Playfair, and Fira Code text adjustments, tracking `letter-spacing`, double leading `line-height` ratios, and stacked glowing `text-shadow` layers.

### 07 Transforms & Anim Studio
* **Scope**: 2D coordinate spaces (`scale`, `rotate`, `skew`), preset timeline animations (`spin`, `pulse`, `float`), and custom timeline speed duration controls.
